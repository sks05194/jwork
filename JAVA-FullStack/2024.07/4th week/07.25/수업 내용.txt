	css 스타일 기술 형식(문법 = Syntax)
	- 인라인스타일시트(inline stylesheet) : html의 시작태그 안에 style 속성으로 기술하는 방식
	형식: style="속성명: 값; 속성명: 값, ..."
	적용범위 : 해당 엘리먼트만

	- 내부 스타일 시트(internal stylesheet) : html에 <style>태그로 기술하는 방식
	형식: 셀렉터(=스타일을 지정할 태그) { 속성명: 값 ; 속성명: 값; ... }
	적용범위 : <style>태그가 기술된 해당 웹페이지(웹문서 = 웹파일)내에서만

	- 외부 스타일 시트(external stylesheet) : css 파일로 별도로 생성하여 기술하는 방식
	형식: 셀렉터(=스타일을 지정할 태그) { 속성명: 값 ; 속성명: 값; ... }
	적용범위 : 해당 css파일을 참조하는 모든 웹페이지(웹문서 = 웹파일)

	a태그(= 앵커태그):
	- a태그가 감싸고 있는 컨텐츠를 클릭하면 페이지가 이동이 되도록 해주는 기능을 가지고 있는 태그임.
	- 반드시 href라는 속성을 같이 기술하여 사용해야만 페이지 이동이 가능하다.
	- href 속성은 기본으로 가지고 있는 스타일의 천성이 존재한다.

	* href 속성이 가지고 있는 동작 4가지
	1. link : 아무것도 하지 않고 그냥 놔둔 상태 - 파란색 글씨 및 밑줄
	2. hover : a태그에 마우스를 올려놓기만 한 상태 - 파란색 글씨 및 밑줄
	3. active : a태그 위에서 마우스를 누르고 있는 상태(클릭 아님) - 빨간색 글씨 및 밑줄
	4. visited : a태그를 클릭하여 해당 페이지로 이동한 내역이 존재하는 상태(이동 내역은 브라우저의 캐시, 방문 기록 등에 존재함) - 보라색 글씨 및 밑줄

	a태그 기술 형식( = Syntax = 문법)
	<a href="이동할 url 주소값 또는 이동할 태그의 #id값" target="값" style="속성명:값; ... ">텍스트 또는 태그</a>

	target 속성 : 페이지를 이동할 때 창의 형태를 지정하는 속성
	열리는 페이지를 어떠한 형태로 열리게 하는지를 선택하는 속성
	_self : (기본값) 현재 창에서 이동하는 페이지가 열림.
	_blank : 다른창(다른탭)에서 이동하는 페이지가 열림.
	_parent: 부모창에서 이동하는 페이지가 열림(IE의 낮은 버전에서만 적용됨)
	_top: 최고 조상창에서 이동하는 페이지가 열림(IE의 낮은 버전에서만 적용됨)
