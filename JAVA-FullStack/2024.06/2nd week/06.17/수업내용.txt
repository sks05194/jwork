09:40	객체 선언문 형식
	**생성자호출문형식 자료형()
	자료형 객체명 = new 생성자호출문();

	//생성자(Constructor) : 클래스에서 객체를 만들때 (인스턴스화 시킬때), 반드시 처음 한 번만 호출되는 호출문. 반드시 클래스 안에 한 개 이상의 생성자가 존재해야 한다.
	생성자가 호출이 되지 못하면 인스턴스를 만들 수 없다.

	리턴타입을 기술하는 기술부가 절대 존재할 수 없다.
	반드시 클래스명과 이름이 동일해야 한다.(대소문자 가림)
	생성자를 한 개도 기술하지 않으면 기본생성자(default constructor)가 숨어져 있다.
	// 기본생성자 형식 : public 클래스명( ){ }
	기타생성자를 생성시 기본생성자가 소멸된다.

5교시
14:30	접근제어자 : public > protected > default > private

	클래스(파일 단위) : 속성들과 기능들을 한꺼번에 묶어놓은 집합체
	자바에서 만들 수 있는 최소 파일 단위
	공통적으로 처리해야하는 기능들이나 속성들을 묶어놓는 단위 = 모듈

	패키지(디렉토리 단위, 폴더 단위) : 클래스급 이상의 파일을 한꺼번에 묶어 놓은 집합체
	- 클래스가 소속되어 있는 집합명

	// 패키지: 나의 클래스의 소속을 명시
	// 형식: package 패키지 경로만;

	// 패키지가 다른 클래스로 객체를 생성하는 경우는
	// 해당 클래스의 소속을 명시해 주어야 한다.
	// 명시하는 형식 : import 패키지경로포함.클래스명까지;
	import 패키지A.*; => 패키지A에 있는 모든 클래스를 불러온다.
	
	클래스의 접근제어자를 적지 않았을 경우 default를 사용한 것으로 간주한다.
