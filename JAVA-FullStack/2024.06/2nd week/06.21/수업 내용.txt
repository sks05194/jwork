5교시
15:10	this 키워드 : this 키워드를 포함하고 있는 클래스 복제본( = 인스턴스 = 주소값), 생성자 선언문, 메소드 선언문 안에 순서 상관없이 기술 가능. (static 메소드 선언문에 사용 불가)

	this( 매개인자_값/*생략가능*/ ) 생성자 호출문 : this 안의 다른 생성자를 호출할때 사용. 생성자 선언문의 가장 맨 첫번째 줄에만 기술 가능.

	super 키워드 : super 키워드를 포함하고 있는 클래스 복제본( = 인스턴스 = 주소값)이 상속받은 부모 클래스 복제본( = 인스턴스 = 주소값)을 의미. 생성자 선언문, 메소드 선언문 안에 순서 상관없이 기술 가능. (static 메소드 선언문에 사용 불가)

	super( 매개인자_값/*생략가능*/ ) 생성자 호출문 : 부모의 생성자 호출문. 자식의 생성자 선언문의 가장 맨 첫번째 줄에만 기술 가능.

	만약 부모클래스의 기본생성자가 기타생성자로 인해 소멸되었을 경우 자식클래스의 생성자에 부모클래스의 기타생성자를 기입해주어야 한다.
