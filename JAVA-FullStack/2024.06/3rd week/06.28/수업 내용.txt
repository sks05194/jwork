    - 나에게 발생한 에러를 내가 직접 처리하는 방식
        try {
            // 실행코드들...;
        } catch(/*예외를_처리할_클래스로_만든_매개변수_선언문*/) {
            // try 블록 내에서 발생한 에러를 처리하는 코드들...;
        } finally{ // finally 생략 가능
            // try 블록 내에서 에러가 발생되든 발생이 되지 않든 반드시 처리해야하는 코드들...;
        }
    - 나에게 발생한 에러를 나를 호출한 곳에서 처리하도록 책임 전가하는 방식 : 메소드 선언문의 ()뒤에 throws를 이용하여 선언함
        [접근제어자] [기타제어자] 리턴타입 메소드명( [매개변수 선언문들... ] ) throws 예외처리클래스명1, 예외처리클래스명2, ... { ... }
