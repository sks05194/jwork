12:15	반복구조
	- 반복문: 어떠한 연산을 여러번 실행해야 하는 경우 사용, 연산의 흐름을 제어하는 제어문 중의 하나.
	(for문, while문, do~while문)

	for문 선언문 형식 : for(초기값 설정식(반복 시작값 설정); 조건식(반복을 종료하기 위해 설정); 증감식(후위연산자 사용 권장)){ /* Do Something...  */ }
	초기 설정 -> 조건식 -> 실행문 -> 증감식 -> 조건식 -> 반복 또는 종료

	for문 내의 초기값 설정식에서 선언된 변수는 for문을 빠져나가면 메모리에서 삭제된다.(변수의 생명주기)
